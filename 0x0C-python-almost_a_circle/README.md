This project will help us be ready for the AirBnB project.

In this project, we will review everything we saw about Python:

* Import
* Exceptions
* Class
* Private attribute
* Getter/Setter
* Class method
* Static method
* Inheritance
* Unittest
* Read/Write file

We will also learn about:
* args and kwargs
* Serialization/Deserialization
* JSON
